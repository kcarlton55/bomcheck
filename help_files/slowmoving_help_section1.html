Introduction

Parts nos. from slow moving inventory can be compared with part nos. of your current project in order to look for possible substitions of parts in order to use up inventory.  File names of BOMs form SolidWorks and SyteLine end with _sw.csv and _sl.xlsx respectively.  By adding a third file to the mix ending with _sm.xlsx, which contains a BOM of slow moving parts, the comparison can be made by clicking the 2nd run button on the menu bar; i.e, the green triangle icon.

Obtaining a slow moving (SM) BOM

Go to https://app.powerbi.com/, then click PC Indiana Dashboards > Inventory (on the left side menu) > Inventory Forecast.  This will lead to a page looking like this:

At the upper right is the "View" drop down menu.  From that pick "Actual size".  Next, from the "More options" menu (the three dots, ...), also at the upper right, pick "Export data" and from the menu that pops up choose "Data with current layout".  Save Excel file to the folder of your choice.  Name the file anything you want, except the name should end with "_sm.xlsx".  For example: slowmoving_sm.xlsx.  (If you don't resize to "Actual size", then the "More options" menu is difficult to access.)

Menu options

You'll notice on the top menu bar a number of icons and buttons. These are:

  1. Run bomcheck button:  This button runs the bomcheck program in order to compare _sw BOMs to _sl BOMs.
  2. Run comparator button:  The button runs the comparator in order to compare _sw/_sl BOMs to parts with in the _sm BOM
  3. The "clear", X, button:  Delete all entries from bomcheck's drag/drop list so that you can repopulate it with other files that you want to examine.
  4. folder button:  Open the last folder you used from which you dragged files from.
  5. filter: This is a mask by which you can select what part nos. the comparator compares.  This filter is examined in more detail on page two.
  6. % similarity:   When the comparator is run, you'll see a column named "similar" in the results. The number there indicates the similarity of the text from the sw/sl descriptions to those from the sm BOM.  This similarity does not garantee that the sm part can replace a part from the sw/sl BOM, but rather allows the user to examine first those parts that are more likely to be alternate part candidates.  You can enter a number in this field to not show lesser scoring parts.  For example, if you enter 50, then only scores of 50% and above will show in the results.
  7. age:   In the comparator's results is a column named "age". This is how long ago that the part has last been used (days).  If, for example, 60 is entered, then on parts that have been sitting in the warehouse more that 60 days will be shown.
  8. switches:   Check boxes by which you can further filter results.
    1. Include "No Demand" pns. The default is to filter out parts where there is a Demand for those parts.
    2. Include "On Hand" pns. The default is to filter out "Qty On Hand" parts that are zero.
    3. Do not show pns listed in the drop list.  Do not show SW/SL/SM parts listed in the drop list (list is shown in "settings").

Note: When you hover the mouse over any of this buttons, you'll see a description of the button at the bottom of the main window.

Settings

If you go to File > Settings, you will find a "drop list" and an "exceptions list".  Any part number in the drop list will not appear in results of the comparison of slow moving parts.  In the example above, any part nos. that begin with 10, 24, 25, etc. will not be shown.  These numbers are pump, paint, stickers, etc. part numbers.  Exceptions to the list are in the exceptions list.  For example, 2672 is for VFD electrical components.
